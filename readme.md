## 回溯算法

回溯算法在树中就是dfs
```
backtracking() { 
    if (终止条件) {
         存放结果;
     }
     for (选择：选择列表（可以想成树中节点孩子的数量）) {
        递归，处理节点;
         backtracking(); 
        回溯，撤销处理结果 
    } }
```
