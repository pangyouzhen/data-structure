# 常见例子的状态转移方程
## 一维情况
1）最长递增子序列： 
`dp[i]` 代表的是以 `x[i]` 为最后一个值的最大长度

```
dp[x] = max(dp[x], dp[y] + 1) if y < x  && nums[x] > nums[y]
result = max(dp)
```

2）最大子序列和
基本思想：如果上一个值为正，则加到现在的这个上
```
dp[i] = dp[i-1] + nums[i] if dp[i-1] > 0
result = max(dp)
```

## 二维情况 -> 矩阵
2) 最长公共子序列：
子序列：可以不连续。 子串：必须连续

dp[0][i] = 0
dp[j][0] = 0
# 这两条表示 第一行和第一列均为0
dp[i][j] = dp[i-1][j-1] + 1 if str[i] == str[j]
# 如果相同从斜对角线上+1
dp[i][j] = max(dp[i-1][j],dp[i][j-1]) if str[i] != str[j]
# 如果不同从 上一行或者上一列对应值+1
