## 回溯问题的框架

``` 
backtracking() { 
    if (终止条件) {
         存放结果;
     }
     for (选择：选择列表（可以想成树中节点孩子的数量）) {
         one_ans.append(value)--> 这里是纵向的;
         backtracking()
         one_ans.pop() -->(回溯) 纵向的再折回上一个节点 
    } }
```

三个要素

1. 路径：也就是已经作出的选择
1. 选择列表：当前可以做的选择
1. 结束条件：到达决策树的底层，没法进行选择的问题
