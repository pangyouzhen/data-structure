## 回溯问题的框架

``` 
result = []
def backtrack(路径, 选择列表):
    if 满足结束条件:
        result.add(路径)
        return

    for 选择 in 选择列表:
        做选择
        backtrack(路径, 选择列表)
        撤销选择

```

三个要素
1. 路径：也就是已经作出的选择
1. 选择列表：当前可以做的选择
1. 结束条件：到达决策树的底层，没法进行选择的问题
